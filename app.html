Help

Global keybindings
  • Undo: Ctrl + Z
  • Redo: Ctrl + Shift + Z or Ctrl + Y
  • Cancel/Close: Esc
  • Ok: Enter
  • Save: Ctrl + S
  • Save as: Ctrl + Shift + S
  • Open QRt: Ctrl + O
  • Edit QRt: Ctrl + E
Drawing keybindings
  • Draw with first color: LMK
  • Draw with second color: RMK
  • Circle instead of ellipse: Shift
  • Draw circle by middle pointer: Ctrl
Charmap keybindings
  • Enter text in edit mode: Ins
